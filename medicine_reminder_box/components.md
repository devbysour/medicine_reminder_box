# Components List for Medicine Reminder Box

| Component                | Quantity  | Description                       |
| ------------------------ | --------- | --------------------------------- |
| ESP32 DevKit V1 (30-pin) | 1         | Main microcontroller board        |
| DS3231 RTC Module        | 1         | Real-time clock module            |
| 0.96" OLED Display (I2C) | 1         | Display for time & reminders      |
| Active Buzzer            | 1         | Audible alert                     |
| IR Sensor                | 1         | Detects hand opening medicine box |
| Push Buttons             | 3         | Set hour, minute, switch reminder |
| Jumper Wires             | As needed | Connections                       |
| Breadboard               | 1         | Prototype connections             |
| USB Cable                | 1         | Power & programming               |
