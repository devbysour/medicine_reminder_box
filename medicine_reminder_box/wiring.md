# Wiring Connections for Medicine Reminder Box

| Component     | ESP32 Pin |
| ------------- | --------- |
| OLED VCC      | 3.3V      |
| OLED GND      | GND       |
| OLED SDA      | GPIO 21   |
| OLED SCL      | GPIO 22   |
| RTC VCC       | 3.3V      |
| RTC GND       | GND       |
| RTC SDA       | GPIO 21   |
| RTC SCL       | GPIO 22   |
| Buzzer +      | GPIO 27   |
| Buzzer -      | GND       |
| Switch Button | GPIO 13   |
| Hour Button   | GPIO 12   |
| Minute Button | GPIO 14   |
| IR Sensor     | GPIO 32   |
